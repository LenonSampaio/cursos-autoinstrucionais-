Página de Exemplo

Esta é uma página de exemplo que demonstra como reutilizar os componentes de navegação (navbar e sidebar) em qualquer página do projeto.

Os componentes são carregados automaticamente via JavaScript, mantendo consistência e facilitando a manutenção do layout em todo o site.

Como Usar os Componentes

  1. Adicione os containers HTML nos locais apropriados
  2. Inclua o script component-loader.js no final do body
  3. Os componentes serão carregados automaticamente
